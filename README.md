# NSI-projet

    Projet NSI – DealTrack
    Objectif du projet

Ce projet a été réalisé dans le cadre des projets de fin d’année en NSI.

L’objectif principal est de développer une application Python complète permettant de :

Permettre de trouver un deal rentable et de comparer les diffrents prix
traiter et exploiter des données réelles
proposer une interface utilisateur fonctionnelle
appliquer plusieurs notions vues en cours (algorithmique, POO, fichiers, etc.)

🧰 Fonctionnalités principales

    L’application permet de :
    
    Trouver des deal sur des jeu choisi 
    Gerer ses deals
    Comparer les prix

⚙️ Technologies et bibliothèques utilisées

    Ce projet utilise les technologies suivantes :
    
    Python 3
    [requests] 
    [tkinter / customtkinter]
    [JSON]

📊 Fonctionnalités techniques intégrées

    Ce projet respecte les exigences en intégrant :
    
    
    ✔ Programmation orientée objet (classes)
    
    ✔ Gestion de fichiers ( JSON )
    
    ✔ Interface utilisateur (console / graphique / web)
    
    ✔ Gestion des erreurs (try/except)
    
    ✔ Traitement de données
    
    ✔ Algorithme(s) spécifique(s) : [recherche / tri ]
    
    ✔  utilisation d’une API 
    
🚀 Installation et lancement

    1. Cloner le projet
    
    git clone https://github.com/Aaron-78-dpt/NSI-projet.git
    cd NSI-projet
    
    2. Installer les dépendances
    
    pip install -r requirements.txt
    
    3. Lancer le programme
    
    python main.py
